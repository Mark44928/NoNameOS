# Contributing to NoNameOS

Thanks for your interest in contributing! Here's how to get started.

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/my-feature`)
3. **Commit** your changes (`git commit -m "Add my feature"`)
4. **Push** to the branch (`git push origin feature/my-feature`)
5. **Open** a Pull Request

## Development Setup

```bash
git clone https://github.com/YOUR_USERNAME/NoNameOS.git
cd NoNameOS
g++ -O3 NoNameOS.cpp -o nonameos
./nonameos
```

## Guidelines

- Follow the existing code style
- Keep changes focused and minimal
- Test your changes before submitting
- Update documentation if needed

## Ideas for Contributions

- Add new games (snake, minesweeper, etc.)
- Add new shell commands
- Improve the AsciiDash engine
- Add color themes
- Fix bugs

## Reporting Bugs

Use the [Bug Report](https://github.com/Mark44928/NoNameOS/issues/new?template=bug_report.md) template.

## Requesting Features
